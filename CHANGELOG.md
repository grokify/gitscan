## [v0.1.0] - 2026-01-17

### Highlights

- CLI tool to scan multiple Git repositories and identify repos needing attention
- Zero external dependencies - uses only Go standard library

### Added

- Repository scanner that checks for uncommitted changes using `git status --porcelain`
- Replace directive detection in go.mod files (both single-line and block format)
- Module name mismatch detection comparing go.mod module path with directory name
- Dependency filtering with `-dep` flag to find repos depending on a specific module
- Recursive nested go.mod search with `-recurse` flag for monorepo support
- Progress bar with percentage, count, and current repo name during scanning
- Two output formats: human-readable list (default) and compact markdown table
- Summary statistics showing counts of uncommitted changes, replace directives, and mismatches
- Tilde expansion for home directory paths
- Support for positional argument or `-dir` flag for directory path
