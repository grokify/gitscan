## [v0.2.0] - 2026-02-14

### Highlights

- New `order` subcommand for topological dependency ordering
- Time-based filtering with `-since` flag
- Parallel scanning with 50% faster performance

### Added

- New `order` subcommand to show repos in dependency order (update dependencies first) ([`104f2a2`](https://github.com/grokify/gitscan/commit/104f2a2))
- Topological sort using Kahn's algorithm for dependency-aware ordering ([`7f8ff17`](https://github.com/grokify/gitscan/commit/7f8ff17))
- Time-based filtering with `-since` flag supporting `7d`, `2w`, `1m` duration formats ([`33af0ef`](https://github.com/grokify/gitscan/commit/33af0ef))
- Transitive dependency tracking with `-t` flag to include repos that depend on modified repos ([`7f8ff17`](https://github.com/grokify/gitscan/commit/7f8ff17))
- Unpushed commit detection with `-u` flag to filter repos needing push ([`7f8ff17`](https://github.com/grokify/gitscan/commit/7f8ff17))
- Internal dependency display showing `(depends on: x, y)` in output ([`33af0ef`](https://github.com/grokify/gitscan/commit/33af0ef))
- Latest modification time tracking per repository ([`7f8ff17`](https://github.com/grokify/gitscan/commit/7f8ff17))
- Parallel scanning using goroutine worker pool (uses GOMAXPROCS workers) ([`7f8ff17`](https://github.com/grokify/gitscan/commit/7f8ff17))
- Lazy evaluation of expensive operations (mod time, unpushed checks) ([`7f8ff17`](https://github.com/grokify/gitscan/commit/7f8ff17))
- Aligned output formatting with consistent column widths ([`33af0ef`](https://github.com/grokify/gitscan/commit/33af0ef))

### Changed

- Migrated CLI from standard `flag` package to Cobra for better UX ([`33af0ef`](https://github.com/grokify/gitscan/commit/33af0ef))
- Added short flags: `-d` (dir), `-f` (format), `-r` (recurse), `-s` (since) ([`33af0ef`](https://github.com/grokify/gitscan/commit/33af0ef))
- Output now shows numbered list with alphabetical sorting ([`33af0ef`](https://github.com/grokify/gitscan/commit/33af0ef))
- Refactored scanner package with `ScanOptions` for configurable behavior ([`7f8ff17`](https://github.com/grokify/gitscan/commit/7f8ff17))

### Performance

- Optimized git status to single command per repo (was 2-3 commands) ([`6e88de7`](https://github.com/grokify/gitscan/commit/6e88de7))
- Scan time reduced by ~50% (6.7s â†’ 3.4s for 600 repos) ([`6e88de7`](https://github.com/grokify/gitscan/commit/6e88de7))

### Dependencies

- Added `github.com/spf13/cobra` for CLI framework ([`33af0ef`](https://github.com/grokify/gitscan/commit/33af0ef))
- Added `github.com/grokify/mogo/fmt/progress` for progress bar rendering ([`9c27c2a`](https://github.com/grokify/gitscan/commit/9c27c2a))

### Documentation

- Updated README with order subcommand, new flags, and performance section ([`e319764`](https://github.com/grokify/gitscan/commit/e319764))
- Added RELEASE_NOTES_v0.2.0.md ([`e319764`](https://github.com/grokify/gitscan/commit/e319764))

## [v0.1.0] - 2026-01-17

### Highlights

- CLI tool to scan multiple Git repositories and identify repos needing attention
- Zero external dependencies - uses only Go standard library

### Added

- Repository scanner that checks for uncommitted changes using `git status --porcelain` ([`97d0739`](https://github.com/grokify/gitscan/commit/97d0739))
- Replace directive detection in go.mod files (both single-line and block format) ([`97d0739`](https://github.com/grokify/gitscan/commit/97d0739))
- Module name mismatch detection comparing go.mod module path with directory name ([`97d0739`](https://github.com/grokify/gitscan/commit/97d0739))
- Dependency filtering with `-dep` flag to find repos depending on a specific module ([`97d0739`](https://github.com/grokify/gitscan/commit/97d0739))
- Recursive nested go.mod search with `-recurse` flag for monorepo support ([`97d0739`](https://github.com/grokify/gitscan/commit/97d0739))
- Progress bar with percentage, count, and current repo name during scanning ([`97d0739`](https://github.com/grokify/gitscan/commit/97d0739))
- Two output formats: human-readable list (default) and compact markdown table ([`97d0739`](https://github.com/grokify/gitscan/commit/97d0739))
- Summary statistics showing counts of uncommitted changes, replace directives, and mismatches ([`97d0739`](https://github.com/grokify/gitscan/commit/97d0739))
- Tilde expansion for home directory paths ([`97d0739`](https://github.com/grokify/gitscan/commit/97d0739))
- Support for positional argument or `-dir` flag for directory path ([`97d0739`](https://github.com/grokify/gitscan/commit/97d0739))
